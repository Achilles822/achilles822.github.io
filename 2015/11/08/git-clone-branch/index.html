git clone其他分支的方法

Posted on 2015-11-08

“为了用上这个主题稍微去学了一下git。”

直接上干货好了：

clone xxx.git //把所需项目clone到本地

git checkout -b "要下载的分支名"  //转换到该分支，注意引号要去掉

git pull xxx.git "要下载的分支名" //把代码拉到本地

似乎渐渐地喜欢上用git了，慢慢积累吧~ ：)
