windows环境Ruby、Sass、Compass环境配置

Posted on 2016-11-16

今天介绍下Win下如何配置Ruby、Sass、Compass：

安装Ruby

在Ruby官网下载安装RubyInstaller。

安装时记得勾选“add to PATH”选项

最好把git也装上，其命令行工具更好用。

更换成境内的源

gem sources --remove https://rubygem.org/

gem sources -a https://gems.ruby-china.org/

如果遇到SSL证书问题，可将https协议换为http协议

使用以下命令检查源是否正确：

gem sources -l

安装Sass

gem install Sass

检查Sass版本

sass -v

安装Compass

gem install Compass

检查Compass版本

compass -v

Compass创建项目

compass create your-project-name

Compass设置监听

open your project PATH

compass watch
